PQ-AT — демонстрация

Прототип постквантового токена доступа из магистерской диссертации. Документация API — Swagger UI.

1. Логин

POST /auth/token с grant_type=password. Доступные seed-учётки: alice / alice-password-2026, bob / bob-password-2026, charlie / charlie-password-2026, dave / dave-password-2026. 







  
2. Обновление по refresh-токену

  

    POST /auth/refresh с grant_type=refresh_token. Refresh подставляется
    из результата логина; rotation инвалидирует старый.
  

  

    
    
  

  






  
3. Отзыв токена

  
POST /auth/revoke. Подсказка типа берётся из claim kind.

  

    
    
  

  






  
4. Декодирование (без проверки подписи)

  
Клиентский разбор header и claims из текстового токена. Подпись не сверяется.

  

    
    
  

  






  
5. Публичные ключи (JWKS)

  
GET /.well-known/pq-jwks.

  
  






  
6. Метаданные сервера авторизации

  
GET /.well-known/oauth-authorization-server (RFC 8414).
